七,static关键字
	中文:静态的
	静态的属性和方法,是类的
	也就是static修饰的属性,或者是方法
	是类的
	!!!!   不是依赖于对象的
	
	1,静态方法中不能使用非静态的属性
	2,静态方法或属性,可以理解为共享的状态
		对每个对象操作的都是一个相同的属性或方法
			注意是相同的,不是两个东西一模一样的相同
			指的是在内存中,就存在一个!!
	3,静态的属性和方法,可以使用类名直接调用
	eg:
		static int a;
		static void show(){}
		
		类名.a;
		类名.show();
